# 设计器

- 新建页面或区块时校验和自动格式组件名称
- 样式设置面板
- 属性设置器支持多级对象/数组
- API管理支持请求设置配置

# UI组件库

- XDialogForm 弹窗表单
- XChart 数据图表
- XUpload 上传组件
- XTabs 选项卡
- XToolbar 工具条
- XTable 表格组件
- XCard 卡片组件
- XList 数据列表
- XDataItem 数据项
- XDataCard 数据卡片
- XDataPicker 数据选择器
- XLogin 登录组件（支持账号密码、短信验证码、扫码）登录
