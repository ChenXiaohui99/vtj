# numeral 数字格式化  



| 函数名       | 描述                          | 类型     | 参数                                                                       | 返回值   |
| ------------ | ----------------------------- | -------- | -------------------------------------------------------------------------- | -------- |
| numberFormat | 数字格式化，默认2位小数       | function | (`val:number`, `format<0.00>?:string`)                                     | `number` |
| toFixed      | 保留小数点，默认2位           | function | (数值`val:number`, 小数位数`format<2>:number`,是否四舍五入`round:boolean`) | `number` |
| numeral      | [文档](http://numeraljs.com/) | -        | -                                                                          | -        |
