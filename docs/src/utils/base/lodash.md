# Lodash

内置 Lodash 工具函数

## 数据类型校验

| 函数名     | 描述                     | 类型                    | 参数         | 返回值     |
| ---------- | ------------------------ | ----------------------- | ------------ | ---------- |
| isString   | 判断数据类型是否字符串   | `(val: any) => boolean` | 需检测的数据 | 是否字符串 |
| isFunction | 判断数据类型是否函数类型 | `(val: any) => boolean` | 需检测的数据 | 是否字函数 |
